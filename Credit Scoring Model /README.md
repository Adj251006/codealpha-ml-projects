# 📘 Task 1: Credit Scoring Model

This project is part of the **Code Alpha Machine Learning Internship**.

---

## 📌 Project Overview
The Credit Scoring Model predicts the creditworthiness of individuals based on historical financial and demographic data.  
It helps financial institutions assess lending risk using machine learning techniques.

---

## 🎯 Objective
- Analyze customer financial data  
- Perform data preprocessing and visualization  
- Train machine learning models  
- Predict credit scores accurately  

---

## 📂 Dataset Description
The dataset contains information such as:
- Age  
- Income  
- Loan Amount  
- Credit History  
- Employment Status  
- Payment Behavior  

*Dataset is used for educational purposes only.*
